使い方

Finder で選択されているファイルやフォルダに対して実行されるので、Dock や Script Menu、何らかのランチャーに登録して起動してください。

Finder で名前を一括して変更したいファイル群を選択して、（何らかのランチャー上から）PowerRenamer を起動します。すると以下のようなダイアログが表示されます。

ここで、Search Text のテキストフィールドに、ファイル名中の置き換えたい文字列を指定しください。Replace Text のところで、どの文字列に置き換えるかを指定します。

Search Text のポップアップメニューで、置き換えたい文字列の条件を設定する事ができます。

  • Contain
    • 置き換えたい文字列が名前のどこかに含まれている
  • Beginning
    • 置き換えたい文字列が名前の先頭にある
  • End
    • 置き換えたい文字列が名前の末尾にある
  • Regular Expression
    • 正規表現で指定（使える正規表現についてはこちらをご参照ください。）

そして、OK ボタンを押せば選択したファイルの名前に含まれている文字列が全て置き換えられます。

ちたみに、OK ボタンを押した時点で Finder で選択されている物を対象にしますので、PowerRenamer が起動した後でも、名前を変更する対象を変える事ができます。

Search Text に「Beginning」もしくは「Endding」の条件が設定されている場合、Search Text のテキストフィールドに文字を代入せずに、「Replace Text」のところだけ、文字を入力して実行したすると、先頭や末尾に文字列を付け加えることができます。
